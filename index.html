Movie Me!

- A fullstack web app using the MERN stack and a movie database API.

Movie Me! is a fullstack web app that uses a movie database API by The Movie Database (TMDB). A user can query hundreds of thousands of movies by selecting from 4 different filters that utilize API presets with algorithms I wrote to optimize search results. A user can also create a watchlist which they can view and edit at any time.
