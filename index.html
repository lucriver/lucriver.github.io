Memz

-A media focused personal full stack web app, using GraphQL, Firebase authentication and more.

Memz is a personal full stack web app that allows a user to upload pictures, videos and post journals. The app is easily scalable into a full fledged social media application. Memz uses React and Typescript for the front end, Google firebase for authentication, Express, Apollo Server and GraphQL for the backend, and PostgreSQL for the database.
