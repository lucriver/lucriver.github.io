Memz

-A media focused, personal, full stack web app, using GraphQL, Firebase authentication and more.

Postal Parser

-An example of a school project that highlights my C++ and coding proficiency.

Postal Parser is an example of a recent project completed in one of my classes at University. This program takes a comma separated file of zip codes (In our case, over 40,000 records long), with each record containing the relevant cordinates, state, city, etc.. This program formats this raw CSV file into a blocked sequence set file of records, with each block number being a primary key correlating to an index file. A user can insert records, delete them, and find coordinate maximums & minimums for each zip code in each state. All completed using buffer classes that I wrote.
